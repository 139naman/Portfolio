Starting My Java Journey

I chose Java as my main language mainly because of OOP. I wanted to learn a language that wasn’t overly complex and strict like C++, but also not too abstract or easy to start with like Python. Java felt like the right middle ground. So far, I’ve been consistently learning core Java concepts like OOP, exception handling, encapsulation, and my personal favourite — inheritance. I plan to start DSA once I feel fully comfortable with Java syntax and foundational concepts.

Why I'm Learning Spring Boot

As I explored backend development, Spring Boot felt like the most suitable and reliable framework to learn with Java. I really like its simplicity, structure, and the way it helps in building scalable applications efficiently

Learning Through Building

More than just improving my resume, I want to learn by building projects that solve real problems. I find myself far more motivated and excited when I understand the purpose behind what I’m building and why it actually matters.
